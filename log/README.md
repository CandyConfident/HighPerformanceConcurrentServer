## 日志系统
&emsp;&emsp;日志系统包括pr模块和log模块，其中pr模块以区分日志级别的形式简单封装了printf；log模块实现了同步和异步日志，可选择向文件或者屏幕输出日志。
### log模块
> * 日志类为单例模式
> * 同步日志时使用日志的线程竞争锁向文件写入内容
> * 异步日志时使用日志的线程向阻塞队列写入内容
> * 异步日志时由单独的日志线程读取阻塞队列内容写入文件
> * 日志文件按天分类
> * 日志文件限制最大行数
